# WireGuard Setup – Windows

## Step 1: Download WireGuard
Download and install WireGuard from:
https://www.wireguard.com/install/

## Step 2: Import Configuration
1. Open WireGuard
2. Click **Import tunnel → From file**
3. Select the `.conf` file sent to your email

## Step 3: Connect
Activate the tunnel. You are now connected.
